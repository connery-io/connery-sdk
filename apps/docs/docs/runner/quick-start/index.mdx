# Quickstart

This tutorial will guide you on how to use Connery in about 30 minutes.

You will learn how to:

1. Set up your runner on GitHub Codespaces.
2. Install the Gmail plugin on the runner and configure it.
3. Use the Slack App client to send emails from your Gmail account using natural language.
